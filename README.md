# Image Slider

A simple image slider (carousel) built using HTML, CSS, and JavaScript.

## 📸 Features

- **Smooth sliding transitions**
- **Responsive layout**
- **Manual navigation with arrows**
- **Easy to customize with your own images**

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
