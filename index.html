Image Slider

Quick description of the project

This project features a responsive, minimalist slider that cycles through a set of bird images. Using pure HTML, CSS, and JavaScript, I crafted an intuitive layout where the user can navigate forward and backward through the gallery using arrow controls.

Each transition between images is enhanced by a soft fade effect, made possible through dynamic class manipulation and smooth CSS opacity transitions. The image is wrapped in a clean container with object-fit styling, ensuring graceful scaling without distortion. Everything’s optimized for both desktop and mobile, keeping the experience fluid no matter the screen size.
